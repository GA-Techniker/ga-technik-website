# GA-Technik Website

Professionelle Website für GA-Technik - Ihr Spezialist für MSR (Messen, Steuern, Regeln) und Gebäudeautomation.

## Live
- **Domain:** www.ga-technik.com
- **Deployed on:** Railway

## Technologie
- Next.js 14 mit App Router
- TypeScript
- Tailwind CSS
- Static Export

## Seiten
- Startseite
- Über uns
- Leistungen
- News & Insights
- Kontakt

## Entwicklung
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
